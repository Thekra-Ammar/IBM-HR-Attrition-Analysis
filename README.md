## 📌 Project Overview
This project analyzes employee attrition using the IBM HR Analytics dataset.

## 🎯 Objectives
- Identify key factors influencing employee attrition
- Build a predictive model
- Segment employees using clustering

## 🛠 Tools & Technologies
- Python (Pandas, Scikit-learn)
- Power BI
- Machine Learning

## 🤖 Models Used
- Random Forest (prediction)
- K-Means Clustering (segmentation)
- PCA (visualization)

## 📊 Key Insights
- Overtime strongly increases attrition risk
- Low job satisfaction leads to higher turnover
- Employees with low tenure are more likely to leave

## 📈 Results
- Random Forest achieved good prediction performance
- K-Means identified 3 employee profiles
